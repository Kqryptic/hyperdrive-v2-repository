# TheMind Sovereign (v2.0)

## Overview
TheMind v2.0 is a 100% in-house, sovereign intelligence system built on the Director-Subagent architecture.

## Structure
- `/contracts`: Core Solidity anchors for identity and risk.
- `/src/director`: High-level orchestration logic.
- `/src/subagents`: Task-specific execution agents.

## Core Principles
1. **Sovereignty**: 100% in-house code, no legacy dependencies.
2. **Intelligence**: Powered by DeepSeek V4 and Claude Code agentic patterns.
3. **Monetization**: Native Hurdle-to-Revenue (H2R) engine.
